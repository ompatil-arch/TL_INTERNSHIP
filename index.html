Project Documentation

Smart Attendance System (ESP32 + RFID)

Overview: RFID-based attendance system with Google Sheets & Firebase.

Problem: Manual attendance is time-consuming and error-prone.

Objectives: Automate attendance, reduce errors, real-time logging.

Components: ESP32, RFID-RC522, LEDs, Buzzer, Breadboard.

Working: RFID reads UID, ESP32 verifies, logs to Sheets via Apps Script.

Hardware: ESP32, RFID module, jumper wires.

Software: Arduino IDE, Google Apps Script, Firebase.

ESP32 Programming: SPI, WiFi, HTTPClient.

Google Sheets Integration: Apps Script web app.

Firebase: Realtime DB for backup.

Results: 98% accuracy, real-time logging.

Challenges: WiFi connectivity, RFID read range.

Future Scope: Face recognition, mobile app.

Conclusion: Successful and reliable system.
