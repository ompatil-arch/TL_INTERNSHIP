Introduction to some basic video editing software and installation process

OBS Studio

Introduction

OBS Studio is a free and open-source software used for:

  • Screen recording

  • Live streaming

  • Video capture

  • Audio recording

It is widely used by YouTubers, gamers, teachers, and content creators.

Uses

  • Record computer screen

  • Live stream on YouTube, Twitch, Facebook

  • Record tutorials and lectures

  • Add webcam and microphone

  • Create professional streaming layouts

Purpose

The main purpose of OBS Studio is to create high-quality video recordings and live streams without paid software.

Installation Process (Step-by-Step)

Step 1: Open Official Website

Go to:
OBS Studio Official Website

Step 2: Download OBS Studio

  • Click on Windows (or Mac/Linux based on your system).

  • Download will start automatically.

Step 3: Open Installer

  • Open downloaded .exe file.

  • Click Next.

Step 4: Accept License

  • Click I Agree.

Step 5: Install Software

  • Click Install.

  • Wait for installation to complete.

Step 6: Launch OBS Studio

  • Click Finish.

  • OBS Studio opens.

Step 7: Auto Configuration Wizard

  • Select:

    • Optimize for Streaming OR

    • Optimize for Recording

  • Click Next and complete setup.

GIMP is a free image editing software used for photo editing and graphic design.

Uses

  • Logo design

FFmpeg

Introduction

FFmpeg is a powerful command-line multimedia tool used for:

  • Video conversion

  • Audio conversion

  • Recording

  • Streaming

  • Compression

  • Editing media files

Uses

  • Convert MP4 to MP3

  • Compress videos

  • Cut videos

  • Extract audio

  • Record screen

  • Stream media

Purpose

The purpose of FFmpeg is to process multimedia files efficiently using commands.

Installation Process (Windows)

Step 1: Open Official Website

Go to:
FFmpeg Official Website

Step 2: Download Windows Build

  • Click Download.

  • Choose Windows build from recommended providers.

Step 3: Extract ZIP File

  • Extract downloaded ZIP file using WinRAR or 7-Zip.

Step 4: Move Folder

  • Move extracted FFmpeg folder to:
    C:\ffmpeg

Step 5: Add FFmpeg to PATH

  • Search Environment Variables in Windows.

  • Open Edit the system environment variables.

  • Click Environment Variables.

  • Select Path → Edit.

  • Add:
    C:\ffmpeg\bin

Step 6: Verify Installation

Open Command Prompt and type:

ffmpeg -version

If installation is successful, FFmpeg version information will appear.

Conclusion

In this internship session, we learned about different useful software tools such as OBS Studio, GIMP, and FFmpeg. These tools are important for screen recording, image editing, video processing, and multimedia creation. We also understood their installation process and practical uses. This knowledge will help in future technical projects, content creation, and software learning.

Inkscape — Logo Design

Introduction to Inscape and some basic Information

Inkscape is mainly used for creating and editing logos because it provides powerful vector design tools. Logos made in Inkscape can be resized to any size without losing quality, which is very important for printing, branding, websites, and social media

Uses of Inkscape for Logos:

  • Create professional vector logos

  • Design custom shapes and icons

  • Add and edit text styles

  • Use colors, gradients, and effects

  • Export logos in PNG, SVG, PDF, and other formats

  • Create scalable logos for banners, business cards, and websites

Purpose of Inkscape for Logo Design:

The main purpose of Inkscape in logo designing is to help users create high-quality, scalable, and professional logos freely without paid software.

Installation process of Inkscape

Step 1: Open Google and Search Inkscape

Step 5: Complete Installation

Project Documentation

Smart Attendance System (ESP32 + RFID)

Overview: RFID-based attendance system with Google Sheets & Firebase.

Problem: Manual attendance is time-consuming and error-prone.

Objectives: Automate attendance, reduce errors, real-time logging.

Components: ESP32, RFID-RC522, LEDs, Buzzer, Breadboard.

Working: RFID reads UID, ESP32 verifies, logs to Sheets via Apps Script.

Hardware: ESP32, RFID module, jumper wires.

Software: Arduino IDE, Google Apps Script, Firebase.

ESP32 Programming: SPI, WiFi, HTTPClient.

Google Sheets Integration: Apps Script web app.

Firebase: Realtime DB for backup.

Results: 98% accuracy, real-time logging.

Challenges: WiFi connectivity, RFID read range.

Future Scope: Face recognition, mobile app.

Conclusion: Successful and reliable system.
