GitHub - Introduction, Uses, Purpose, and Installation Guide

1. Introduction to GitHub

GitHub is a web-based platform used for version control and software development. It helps developers store, manage, track, and collaborate on coding projects.

2. Uses of GitHub

  • Store source code online

  • Track project changes

  • Collaborate with teams

  • Manage open-source projects

  • Backup important files

3. Purpose of GitHub

The main purpose of GitHub is to improve teamwork and project management in software development.

Configure Git with Your Name & Email

Git needs to know who you are. Run these two commands in your Terminal or Command Prompt:

Step 1: Set your name

git config --global user.name "Your Full Name"

Step 2: Set your email (use the same email as GitHub)

git config --global user.email "youremail@example.com"

Step 3: Check your settings

git config --list

Project Documentation

Smart Attendance System (ESP32 + RFID)

Overview: RFID-based attendance system with Google Sheets & Firebase.

Problem: Manual attendance is time-consuming and error-prone.

Objectives: Automate attendance, reduce errors, real-time logging.

Components: ESP32, RFID-RC522, LEDs, Buzzer, Breadboard.

Working: RFID reads UID, ESP32 verifies, logs to Sheets via Apps Script.

Hardware: ESP32, RFID module, jumper wires.

Software: Arduino IDE, Google Apps Script, Firebase.

ESP32 Programming: SPI, WiFi, HTTPClient.

Google Sheets Integration: Apps Script web app.

Firebase: Realtime DB for backup.

Results: 98% accuracy, real-time logging.

Challenges: WiFi connectivity, RFID read range.

Future Scope: Face recognition, mobile app.

Conclusion: Successful and reliable system.
