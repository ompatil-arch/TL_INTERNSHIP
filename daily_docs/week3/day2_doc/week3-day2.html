Documentation on Firebase

Firebase is a platform by Google that helps developers build, run, and grow web and mobile applications without managing much backend infrastructure.

How to Create Use Firebase?

Step 1: Search Firebase on Google Chrome or Microsoft Edge

Step 2: Click on the link

Step 3: Open console in another tab by clicking on it

Step 4: Click on the Get started by setting up a Firebase project

Step 5: Type name for your project

Allow AI assistance to help It helps us during project by adding more details and ideas

Step 6: After Creating project click on Authentication from Security

Step 7: click on get started

Step 8: Get started with Firebase Authentication by adding your first sign-in method

Step 9: Enable permission and enter you e-mail and save it

Step 10: Now select real time database from database and storage

Step 11: Now click on create database

Step 12: Select Singapore in the below option or any of those which is nearer from your location

Step 13: Select start in test mode and click on enable

Step 14: you will get results like this

Step 15: Now go to settings and select general

Step 16: Now select service accounts

Step 17: Select any Admin SDK configuration snippet and click on generate new private key

Step 18: Now click on database secrets

Step 19: Now copy the key

Step 20: Now go to general and select general

Step 21: Scroll down and select web

Step 22: Add Nickname to your app click on register app

Step 23: Get API key from here

Step 24: Now open Arduino IDE

Step 25: Select Board

Step 26: select COM 3

Step 27: Download ESP32

Step 28: Now write the given code in Arduino IDE

Step 28: now change

1.WiFi_SSID with your mobile hotspot name

2.WiFi_password with your mobile hotspot name

3.change Database URL with your real time database URL

4.Change your database secret with your database secret

Step 29: Now take ESP32 Three Jumper wires and DHT11 and connect it like this

Check the program, connections of wire properly and run the program
