Day 1

Autodesk Fusion — Advanced Features

Some Basics features of Autodesk Fusion

Section 1: Creation of Mobile stand Using Autodesk Fusion

Step 1: Select part design and click on “create”

Step 2: Select a plane

Step 3: Click on sketch to create a model.

Step 4: Create a rectangle of dimensions suitable for your Smartphone

Step 5: Now select the shape and press “E” or extrude option and adjust the distance as per your need.

Step 6: Now select Text option from the create menu.

Step 7 :Now select revolve option from the surface and select bottom axis to revolve the text at 45 degree.

Step 8: Now Select the fillet option from modify and make the sharp edges round

Step 9: Create a small rectangle at a specific location

Step 10: Create a line in the middle and select symmetry option from the constraints.

Step 11: extrude the rectangle at 6mm and round the edges using fillet and add some details

Section 2: How to Engrave text on cylindrical objects

Step 1: Select part design and click on “create”

Step 2: Select a plane

Step 3: Click on sketch to create a model.

Step 4: Create the circle of diameter 100mm

Step 5: Select extrude or type “E” and set thickness to 40mm

Step 6: Now select top surface and click on shell to hollow object from inside and select thickness 1mm

Step 7: now select the front plane and write the text.

Step 8: Select text and extrude it

CONCLUSION

Using Autodesk Fusion for designing a phone stand and engraving text on a cylindrical object helps in learning important 3D modelling skills such as sketching, extruding, modifying shapes, and adding custom text. Fusion 360 provides an easy and accurate way to create both simple and complex models with professional results.

The phone stand project improves understanding of product design and measurements, while the cylindrical text engraving process demonstrates how to customize objects using sketch and emboss tools. Overall, Fusion 360 is a powerful and user-friendly software for creating creative, precise, and practical 3D designs.
